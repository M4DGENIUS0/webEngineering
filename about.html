We have built this semester project to fulfill the requirements given to us by our course instructor. The project demonstrates our understanding and application of web technologies including HTML, CSS, JavaScript, and API integration.

Abstract:
This project, Quick Cook, allows users to discover and generate recipes based on ingredients and images provided by the user. We have integrated the MealDB API to display recipes dynamically and offer a seamless cooking experience.
